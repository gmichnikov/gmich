# Kids Words Data

- **grade2_words.json** - Word list for grade 2 mode. Keys are 5-letter words; values "yes"/"no" indicate if the word can be an answer.
- **grade4_words.json** - Word list for grade 4 mode. Same structure; different yes/no values.
- **wordle_guesses.txt** - Acceptable guess words (one per line). Source: [Wordle-Ta](https://gist.githubusercontent.com/scholtes/94f3c0303ba6a7768b47583aff36654d/raw/73f890e1680f3fa21577fef3d1f06b8d6c6ae318/wordle-Ta.txt). These words are valid to type as guesses but will never be answers unless they appear in the answer set for the chosen difficulty.
