# API Response Files

This directory contains sample responses from The Odds API for inspection and model validation.

Files are generated by running `explore_odds_api.py` (located in this directory)

## Typical Files:
- `sports_list.json` - All available sports and leagues
- `events_basketball_nba.json` - Upcoming NBA games
- `events_soccer_epl.json` - Upcoming EPL matches
- `odds_basketball_nba.json` - NBA odds (h2h, spreads, totals)
- `odds_soccer_epl.json` - EPL odds (h2h, spreads, totals)
- `scores_basketball_nba.json` - Recent/live NBA scores
- `scores_soccer_epl.json` - Recent/live EPL scores
